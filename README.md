Cafe Purworejo 

Sebuah platform sederhana berbasis web untuk membantu masyarakat dan wisatawan menemukan tempat nongkrong (cafe) terbaik di Kabupaten Purworejo. Project ini dibangun sebagai bagian dari latihan pengembangan front-end menggunakan **React** dan **Vite**.

---

### 🌐 Demo Website
Kamu bisa mengakses website secara langsung di sini:  
👉 **(https://rayanhakim.github.io/Cafe-Purworejo/)**

---

## 🚀 Fitur Utama
- **Katalog Cafe:** Daftar cafe populer di Purworejo (Kemuning, Plaosan, Sindurjan, dll).
- **Detail Informasi:** Alamat lengkap, lokasi spesifik, dan link Instagram resmi.
- **Responsive Design:** Tampilan optimal baik di desktop maupun perangkat mobile (HP).
- **Auto-Sort:** Nama cafe otomatis terurut berdasarkan abjad.

## 🛠️ Teknologi yang Digunakan
- **React.js** (Library UI)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **GitHub Actions** (CI/CD Deployment)
